Wildlife
Conservation
Tracker

Wildlife Conservation Tracker (WCT) is a web-based application designed to monitor, manage, and protect wildlife in conservation areas. Its primary focus is to provide a digital ecosystem for recording animal data, supporting conservation efforts, and enhancing environmental awareness. (SDGs 13 & 15).

The system is built with Java for the backend to handle data processing and application logic, while the frontend is developed using HTML and CSS to deliver a clean, structured, and user-friendly interface. This combination ensures the application is reliable, easy to use, and suitable for managing conservation data in a clear and accessible way.
